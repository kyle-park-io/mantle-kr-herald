# 검수 가이드 (review.md)

번역과 채널 문구를 **읽고, 고치고, 승인하는 분**을 위한 문서입니다.

명령어를 외우거나 터미널을 쓸 일이 없습니다. 브라우저 화면 하나에서 전부 끝납니다.
프로젝트가 무엇을 하는지 궁금하면 [`capabilities.md`](capabilities.md)를 보세요.

> 설치·수집·발행은 운영 담당자의 일입니다. 이 문서는 **검수 화면**만 다룹니다.

## 1. 화면 열기

운영 담당자가 검수 서버를 띄워두면 브라우저에서 이 주소를 엽니다.

```
http://localhost:5757
```

화면이 안 뜨면 담당자에게 "검수 서버 켜주세요"라고 하면 됩니다. 직접 띄워야 한다면 터미널에서 이 두 줄입니다.

```bash
pnpm build:web
pnpm serve
```

## 2. 두 가지 검수

화면 위쪽에 모드 전환 버튼이 있습니다.

| 모드 | 무엇을 보나 | 언제 |
|---|---|---|
| **1차 검수 (번역)** | 영어 원문 → 한국어 번역 | 번역이 올라온 직후 |
| **2차 검수 (채널)** | 승인된 번역 → X·텔레그램·카카오 등 채널별 최종 문구 | 1차 승인 후 |

**1차가 먼저입니다.** 1차에서 승인된 번역만 2차로 넘어옵니다.

## 3. 1차 검수 — 번역

왼쪽 목록에서 항목을 고르면 오른쪽에 원문과 한국어가 나란히 나옵니다.

목록 위 필터로 `all` / `translated`(미승인) / `approved`(승인됨)를 걸 수 있습니다. 보통 **`translated`만 보면 됩니다** — 아직 손이 필요한 것들입니다.

1. 한국어 칸을 직접 고칩니다.
2. **`저장`** 을 누릅니다.
3. **`승인 ✓`** 를 누릅니다.

> ⚠️ **`승인 ✓`가 회색이면 저장을 안 한 것입니다.** 편집 중에는 승인 버튼이 잠깁니다. 반대로 고친 게 없으면 `저장`이 잠기고 바로 승인할 수 있습니다.
>
> 다른 항목으로 옮기거나 모드를 바꿀 때 저장하지 않은 편집이 있으면 확인 창이 뜹니다. 창이 떴다면 **저장을 안 한 것**이니 취소하고 저장부터 하세요.

승인하면 그 번역이 예시 모음(few-shot)에 들어가서 **다음 번역의 품질 기준이 됩니다.** 대충 승인하면 그 문체가 계속 재생산되니, 애매하면 승인하지 말고 담당자에게 물어보세요.

## 4. 2차 검수 — 채널 문구

같은 내용이라도 채널마다 문구가 다릅니다. 채널·유형 필터로 좁혀서 봅니다.

| 유형 | 어디로 나가나 |
|---|---|
| `x` | X(트위터) |
| `announcement` | 텔레그램 공지방, 카카오톡 |
| `kol` | KOL 방에 보내는 요청 |
| `pr` | PR 메일 |

고치는 방법은 1차와 같습니다 — 고치고 → `저장` → `승인 ✓`.

승인된 문구는 **`복사`** 버튼으로 그대로 복사해서 실제 채널에 붙여넣으면 됩니다.

## 5. 무엇을 기준으로 보나

채널별 검수 항목이 체크리스트로 정리돼 있습니다. **저장소 안에 있지만 문서 폴더가 아니라 설정 폴더에 있습니다:**

| 채널 | 파일 |
|---|---|
| X 게시물 | `conversion/checklist.x.md` |
| 커뮤니티 공지 | `conversion/checklist.announcement.md` |

날짜·수치·리워드 조건이 원문과 맞는지, 이미지와 본문이 어긋나지 않는지, 규제상 문제될 표현이 없는지를 항목별로 확인합니다.

특히 자주 걸리는 두 가지입니다.

- **지난 일정** — 원문이 "3일 남았습니다"인데 이미 지난 경우가 흔합니다. 발행 시점 기준으로 유효한지 봅니다.
- **참여 유도 표현** — X에서는 `지금 거래하세요` 같은 명령형을 쓰지 않습니다. 공지는 `확인하세요` 정도까지 허용됩니다. 채널마다 기준이 다르니 체크리스트를 따르세요.

## 6. 여기서 하지 않는 것

- **실제 게시** — 이 화면은 문구를 확정할 뿐, X나 텔레그램에 자동으로 올리지 않습니다. 승인된 문구를 복사해서 사람이 올립니다.
- **원문 수집·번역 생성** — 운영 담당자가 돌립니다.
- **`발행 ⬆` 버튼** — 승인된 번역을 문서로 올리는 기능입니다. 저장 모드에 따라 팀 공유 드라이브 또는 로컬 폴더(`output/publish/local/`)에 올라갑니다. 채널 게시가 아닙니다. 담당자가 눌러야 하는 버튼이니 헷갈리면 두세요.
