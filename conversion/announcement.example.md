# 커뮤니티 공지 작성 지침

- 핵심 정보를 가장 먼저 보여줍니다. 제목만 읽어도 주제를 알 수 있게 씁니다.
- 도입부는 1~2줄로 요약하고 바로 핵심 정보로 넘어갑니다.
- 긴 설명 대신 `[소제목]` + 불릿으로 정보를 구조화합니다.
- 공식적이고 중립적인 톤. 해설성 문장과 과장 표현은 지양합니다.
- 꼭 필요한 링크와 CTA만 남깁니다.
